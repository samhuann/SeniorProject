Project Goals

How can we create a simple to use, resourceful, and open source platform for analyzing data from biological assays? I have faced many grueling hours trying to figure out how to analyze data through Excel and struggled quite a bit. Biographer  aims to solve this problem.

Simplicity

The application should be easy to use, without needing downloads, and even the most inexperienced user can utilize it effectively.

Resources

Biographer is an environment where it is easy to access your favorite lab tools. Biographer Toolbox contains a small but useful selection.

Open Source

Biographer is 100% free. All of the code is available for viewing and download on GitHub. Knowledge and convenience is not held back.
