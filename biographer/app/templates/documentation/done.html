One Sample t-test

Use when you have one measurement variable and a theoretical expectation of what the mean should be under the null hypothesis. It tests whether the mean of the measurement variable is different from the null expectation. Returns a p-value and a t-statistic.

Two Sample t-test

Use for two samples when you have one measurement variable and one nominal variable, and the nominal variable has only two values. It tests whether the means of the measurement variable are different in the two groups. Returns a p-value and t-statistic.

Bartlett’s Test for Homoscedasticity

Use when you have one measurement variable, one nominal variable, and you want to test the null hypothesis that the standard deviations of the measurement variable are the same for the different groups. Returns a p-value and Bartlett’s Test Statistic.

One-Way Analysis of Variance (ANOVA)

Use when you have one nominal variable and one measurement variable; the nominal variable divides the measurements into two or more groups. Tests whether the means of the measurement variable are the same for the different groups. Returns a p-value, F-statistic, and a bar graph.

Two-Way Analysis of Variance (ANOVA)

Use when you have one measurement variable and two nominal variables, and each value of one nominal variable is found in combination with each value of the other nominal variable. Tests three null hypotheses: the means of the measurement variable are equal for different values of the first nominal variable; the means are equal for different values of the second nominal variable; and that there is no interaction (effects of one nominal variable don't depend on the value of the other nominal variable). Returns an ANOVA table and a bar graph.

Kruskal-Wallis Test

Use the when you have one nominal variable and one ranked variable. Tests whether the mean ranks are the same in all the groups. Returns a p-value and an H-statistic.

Paired t-test

Use when you have one measurement variable and two nominal variables, one of the nominal variables has only two values, and you only have one observation for each combination of the nominal variables; in other words, you have multiple pairs of observations. It tests whether the mean difference in the pairs is different from 0. Returns a p-value, t-statistic, and a bar graph.

Wilcoxon Signed-Rank Test

Use when you'd like to use the paired t–test, but the differences are severely non-normally distributed.

Tests for One Measurement Variable

Measurement variables are things you can measure. An observation of a measurement variable is always a number. Examples
include length, weight, pH, and bone density. Other names for them include "numeric" or "quantitative" variables.
