Biology Scripts

A collection of biostatistics scripts written in Python with Jupyter Notebook.

This directory is not part of the Flask app.

Installation

Download the ‘biographer’ directory. This is where the app lives.

It is a good idea to create a venv for all the packages. Make sure Python is installed on your machine.
