Exact Test of Goodness-of-Fit

Exact Test of Goodness-of-Fit

Use when you have one nominal variable, you want to see whether the number of observations in each category fits a theoretical expectation, and the sample size is small. Returns a p-value.

Chi-Square Test of Goodness of Fit

Use when you have one nominal variable, you want to see whether the number of observations in each category fits a theoretical expectation, and the sample size is large. Returns a p-value and a chi-square statistic.

Chi-Square Test of Independence

Use when you have two nominal variables and you want to see whether the proportions of one variable are different for different values of the other variable. Use it when the sample size is large. Returns a p-value, chi-square statistic, degrees of freedom, an expected frequencies table, and a bar graph.

Fisher’s Exact Test

Use when you have two nominal variables and you want to see whether the proportions of one variable are different depending on the value of the other variable. Use it when the sample size is small. Returns a p-value, an odd-ratio, and a bar graph.

This test does not take any input from your Excel.

Remember, only two columns allowed, and they must have the same sum.

Tests for Nominal Variables

A nominal variable is a type of variable that represents categories or groups without any inherent order or ranking.
For example, consider a variable called "color" that represents the colors of cars: red , blue , and green . Each color is a category, but
there's no inherent order or ranking among them. Red is not "greater" or "higher" than blue ; they are just different categories.
