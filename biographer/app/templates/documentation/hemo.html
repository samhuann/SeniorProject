Hemocytometer (Cell Counter)

Simple cell counter using the Python TrackPy package.

Quick Start

When you choose a file and upload it, make sure the file is formatted in .png or .jpg. Click ‘Submit’ to begin analysis.

Larger images with more objects will take longer to process.

Upload Image File

The image will be displayed to you after a successful upload.

Attempting to upload anything other than a .png or .jpg file will return an error.

Count Cells

Click the button to begin the counting process. Depending on how many objects there are, it may take from seconds to minutes.
