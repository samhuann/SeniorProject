Limitations

Since this is a Flask web app and not a full fledged data analysis software, it is bound to have some limitations.

You cannot customize your graphs (yet).

You cannot edit the Excel file within the web app.

Data cannot be organized within the app, so you have to keep organized on your computer.

Excel is currently the only supported file type.

Documentation
Introduction

This is a simple web application to analyze statistical data that has been formatted with Excel.

Quick Start

When you choose a file and upload it, make sure the file is formatted in Excel. Choose your file and click ‘Upload’.

Make sure the data is formatted properly with Excel and there are no gaps.

Upload Excel File

The Excel file will be displayed to you after a successful upload.

Attempting to upload anything other than an Excel file will return an error.

Analyze Excel File

You can see a dropdown once the file is successfully uploaded. There options to Transform your data, which included options to transform by functions, normalize, or even apply some biochemistry transforms. There are also options for three kinds of statistical tests . Each dropdown will have a small selection of tests to apply to your data.

Be sure to transform your data before beginning a statistical analysis. Also, some statistical tests require the Excel file to be formatted a certain way.
