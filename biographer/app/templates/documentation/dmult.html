Linear Regression and Correlation

Use when you want to know whether one measurement variable is associated with another measurement variable; you want to measure the strength of the association (r-squared); or you want an equation that describes the relationship and can be used to predict unknown values. Returns a p-value, r-value, slope, intercept, standard error, and a scatterplot+line.

Simple Logistic Regression

Use when you have one nominal variable and one measurement variable, and you want to know whether variation in the measurement variable causes variation in the nominal variable. Returns L, k, x0, and a line on scatterplot.

Spearman Rank Correlation

Use to test the association between two ranked variables, or one ranked variable and one measurement variable. You can also use Spearman rank correlation instead of linear regression/correlation for two measurement variables if you're worried about non-normality, but this is not usually necessary. Returns a p-value, Spearman Correlation Coefficient, and a scatterplot.

Polynomial Regression

Use polynomial regression when you have graphed two measurement variables and you want to fit an equation for a curved line to the points on the graph. Returns r-squared and a line on scatter plot.

Analysis of Covariance (ANCOVA)

Use when you want to compare two or more regression lines to each other; ANCOVA will tell you whether the regression lines are different from each other in either slope or intercept. Returns an ANCOVA table and a scatterplot.

Tests for Multiple Measurement Variables

Measurement variables are things you can measure. An observation of a measurement variable is always a number. Examples
include length, weight, pH, and bone density. Other names for them include "numeric" or "quantitative" variables.
